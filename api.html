released.info

API for developers

I know that you are waiting for a release of your favourite technology.

You can call our API for free.

Get a list of supported products:

Example call:

GET https://released.info/products

Example response:

{
      "products": {
          "java",
          "postgresql"
      }
  }
Get status of product:

Example call:

GET https://released.info/products/java
{
      "product": "java",
      "full_name": "Java 17",
      "released": false,
      "release_date": "2021-10-23",
  }
Rate limiting

Status is updated every hour. Clients are suppose to call our API not more frequently than once per hour.
